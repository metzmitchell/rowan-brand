# Rowan fonts

Public guidance lives in `docs/typography.md`.

The public repo documents the brand fonts but does not ship proprietary font binaries. For local generation, install licensed copies of the Rowan fonts on the machine running the generator.

The quote-card generator checks for `tools/rowan-quote-cards/assets/Times New Roman MT Extra Bold.ttf` and falls back to system fonts when that licensed file is not present.
