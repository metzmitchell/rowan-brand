# Rowan brand assets

Public brand repository for Rowan Builder Marketing.

This repo is the canonical home for public-safe Rowan branding materials: visual guidelines, voice and copywriting rules, logo assets, icon direction, color and typography standards, PDF/carousel generation notes, and the deterministic quote-card/carousel generator.

## Public-safety rule

This repo is brand-only. Do not commit API keys, analytics IDs, customer data, internal workflows, private strategy docs, credentials, or licensed font binaries.
