# Rowan color palette

## Primary palette

| Name | Hex | Use |
|---|---:|---|
| Shell | `#f5f2e8` | Default light background |
| Logo Green | `#485e3d` | Brand green, buttons, sparse sections/cards |
| Pink | `#d4a74f` | Accent / CTA, use sparingly |

## Secondary palette

| Name | Hex | Use |
|---|---:|---|
| White | `#eeede9` | Light accent and subtle background design elements |
| Light Accent | `#d4c7b1` | Warm supporting accent |
| Dark Pink / warm gray | `#575654` | Border/accent support, sparingly |
| Dark Green | `#353934` | Dark backgrounds and high-contrast sections |
| Purple | `#3b3a38` | Accent or muted dark support |
| Darkest Purple | `#19161e` | Image overlays and deepest dark surfaces |

## Background rules

- Shell is the default page and section background
- Dark Green is the main dark-section background
- Darkest Purple is mainly for image overlays
- Logo Green can be used as a section/card background, but sparingly
- Do not introduce new background colors unless the brand system is intentionally extended

## Accent rules

- Pink and Dark Pink are accents only
- Use accent color for a primary CTA or a single emphasis moment
- Most pages should only use pink once or twice
- Purple can be used similarly, but still with restraint
- Avoid pink text on light backgrounds unless contrast is tested and the use is deliberate

## Text contrast

- On dark backgrounds, use Shell for text
- On Shell backgrounds, use dark green family colors for headings and body
- Keep small text high-contrast
- Do not use accent colors for small body copy

## `ROWAN` background wordmark

Use White (`#eeede9`) for a huge low-contrast `ROWAN` background wordmark.

Rules:

- Make it oversized and wide
- Place near the bottom of a section or footer
- Let it crop off page edges
- Keep it subtle, almost barely visible
