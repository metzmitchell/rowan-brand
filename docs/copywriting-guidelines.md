# Rowan copywriting and brand voice

## Core voice

Simple language expressing complex understanding.

Rowan should sound clear, calm, useful, and human. The voice is smart without sounding impressed with itself. It should feel like a sharp advisor explaining something plainly to a capable builder over coffee.

## Always

- Use simple words
- Use active voice
- Write like a human speaks
- Use contractions naturally
- Keep paragraphs short
- Make the structure easy to scan
- Be specific and concrete
- Connect tactical points to the bigger business picture
- Use subtle dry humor when it fits
- End crisply, without a summary footer

## Avoid

- Corporate jargon
- AI-generic phrasing
- Forced humor
- Em dashes
- Overhyped claims
- Fake case studies
- Empty superlatives
- Long abstract intros
- Perfectly symmetrical AI-looking lists
- Question-answer patterns like `Problem? Solution.`
- `It's not X, it's Y` contrast framing
- `TED talk on X`

## Tone by context

### Professional writing

- Clear and direct
- Collaborative
- Calmly confident
- Specific about the real business problem
- Nuanced when the topic is complex

### Quick messages

- Short
- Practical
- Clear action focus
- Minimal personality

### Long-form thought leadership

- Practical insight first
- A little more personality
- Stronger point of view
- Builder/remodeler realities over generic marketing advice

## External marketing rules

- Say `you` more than `we` when addressing prospects
- Make claims concrete and falsifiable
- Use real examples or metrics when available
- Talk about customer outcomes more than company features
- Prefer sentence case for headings
- Write placeholder titles first, then improve them after the piece is written
- Use hedging only when the idea truly needs nuance

## Common replacements

| Avoid | Use |
|---|---|
| utilize | use |
| facilitate | help |
| commence | start |
| numerous | many |
| robust | strong |
| seamless | automatic, simple, or name the actual benefit |
| innovative | describe what is different |
| leverage | use |
| mission-critical | important |
| optimize | improve, tighten, or specify the outcome |

## Five-second test

Before publishing, ask:

> Does this sound like a clear, useful explanation to a smart colleague over coffee?

If not, simplify it.
