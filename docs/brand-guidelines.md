# Rowan brand guidelines

These are the public implementation rules for Rowan Builder Marketing's visual system.

## Design principles

- Calm and spacious: use generous whitespace and breathing room
- High contrast of scale: pair very large titles with smaller support text
- Accent restraint: use accent colors sparingly, usually once or twice per page
- Editorial simplicity: let typography and spacing carry the design
- Practical trust: polished, but not shiny or agency-fluffy

## Visual personality

Rowan should feel calm, clear, sharp, and slightly editorial. The system should look intentional without feeling over-designed.

Good design directions:

- Large serif headlines
- Simple color fields
- Off-white and dark green foundations
- Sparse accent moments
- Thin-line, hand-drawn icons
- Minimal decoration

Avoid:

- Busy gradients
- Stock illustration style
- Tech-startup blue palettes
- Heavy shadows
- Loud accent-heavy layouts
- Generic SaaS polish
- Decorative clutter

## Core palette

- Shell: `#f5f2e8`
- Logo Green: `#485e3d`
- Pink accent: `#d4a74f`
- White: `#eeede9`
- Light Accent: `#d4c7b1`
- Dark Pink / warm gray accent: `#575654`
- Dark Green: `#353934`
- Purple: `#3b3a38`
- Darkest Purple: `#19161e`

See `docs/color-palette.md` for usage rules.

## Typography

- H1-H3: Times New Roman MT Ultra Bold or closest licensed equivalent
- H4 / optional display label: Poppins Ultra Bold
- Body: Livvic Regular
- Buttons: Poppins Regular

See `docs/typography.md` for type usage.

## Buttons

Allowed styles:

- Outlined buttons
- Solid buttons with strong contrast

Guidance:

- On Shell backgrounds, Logo Green is the default button color
- Pink can be used for a primary CTA moment, but sparingly
- On dark backgrounds, use Shell text for contrast
- Button text uses Poppins Regular

## Iconography

- Hand-drawn
- Elegant
- Thin-line
- Minimal
- Dark green on light backgrounds
- Shell on dark backgrounds

See `docs/icon-guidelines.md` for details.

## Imagery and overlays

Dark purple overlays are the primary use of the darkest purple tones.

Use a semi-transparent dark purple overlay, often as a gradient, over images when type needs a readable surface.

## Logo notes

- The icon is the main icon mark
- The no-background icon is a Shell-colored `R`
- The no-background logo is Shell-colored `ROWAN` text
- The background logo is Shell text over brand green

## Implementation checklist

- Use Shell as the dominant light background
- Use Dark Green for dark sections
- Use Logo Green sparingly as a section or card background
- Use pink or darker accent colors only for emphasis or CTA moments
- Use Shell text on dark backgrounds
- Use dark green family text on Shell backgrounds
- Keep icons thin-line and hand-drawn
- Keep layouts spacious
- Avoid new colors unless there is a clear brand reason
