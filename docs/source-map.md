# Source map

This repo consolidates public-safe Rowan brand material from the Knowledgebase and OpenClaw workspace.

## Consolidated sources

- Visual brand guidance from `Team/CRM/Rowan/Rowan Website/Rowan Website/Guidelines/Brand Guidelines.md`
- Logo asset from `Agent Space/Brand Assets/Rowan Logo - Ultra Bold.png`
- Public-safe voice and copywriting guidance from `Team/Processes/Rowan Copywriting/`
- Quote-card/carousel generator from `OpenClaw/glarth/skills/rowan-quote-cards/`
- PDF carousel production notes from the quote-card skill method notes

## Excluded on purpose

- API keys and config
- Analytics and GTM/GA4 implementation details
- Newsletter platform credentials or API notes
- Customer/client data
- Private personal context files
- Tax-document PDF tooling
- Proprietary font binaries
- Local `__pycache__` and generated outputs

## Compatibility bridges

Older Knowledgebase references are kept alive. In particular, the website brand guidelines file remains at its old path with a pointer to this canonical brand folder.
