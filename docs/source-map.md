# Source map

This repo consolidates public-safe Rowan brand material from the Knowledgebase and OpenClaw workspace.

## Consolidated sources

- Visual brand guidance from [Rowan Website Brand Guidelines](https://rowan.build/open/?vault=Rowan&file=Team%2FCRM%2FRowan%2FRowan%20Website%2FRowan%20Website%2FGuidelines%2FBrand%20Guidelines.md)
- Logo assets from [Rowan Brand Assets](https://rowan.build/open/?vault=Rowan&file=Agent%20Space%2FBrand%20Assets%2FREADME.md)
- Public-safe voice and copywriting guidance from [Rowan Copywriting](https://rowan.build/open/?vault=Rowan&file=Team%2FProcesses%2FRowan%20Copywriting%2F)
- Quote-card/carousel generator from the local [Rowan quote-card tool](../tools/rowan-quote-cards/)
- PDF carousel production notes from [Rowan quote-card method notes](../tools/rowan-quote-cards/references/method-notes.md)

## Excluded on purpose

- API keys and config
- Analytics and GTM/GA4 implementation details
- Newsletter platform credentials or API notes
- Customer/client data
- Private personal context files
- Tax-document PDF tooling
- Proprietary font binaries
- Local `__pycache__` and generated outputs

## Compatibility bridges

Older Knowledgebase references are kept alive. In particular, [Rowan Website Brand Guidelines](https://rowan.build/open/?vault=Rowan&file=Team%2FCRM%2FRowan%2FRowan%20Website%2FRowan%20Website%2FGuidelines%2FBrand%20Guidelines.md) remains at its old path with a pointer to the canonical [Rowan Brand Assets](https://rowan.build/open/?vault=Rowan&file=Agent%20Space%2FBrand%20Assets%2FREADME.md) folder.
